# Master Django Template

## Create Development Enviroment

### From the host machine to start the Vagrant enviroment:

```shell
vagrant up
vagrant ssh
```

### Inside the Vagrant enviroment:

```shell
./make-devenv
./run-server
```

### To clean the develepment enviroment:

```shell
./make-clean
```
